Fullstack, fully-typed, battle-tested, yet minimal boilerplate for your TypeScript projects.

Featuring: 
- Full-stack typing
- Docker-based development environment
- OpenAPI-based API
- React frontend apps
- NestJS backend API
- MikroORM for database
- Better Auth for authentication
- Shadcn UI for components
- Tailwind CSS for styling

Used every day at [Lonestone](https://lonestone.io).
